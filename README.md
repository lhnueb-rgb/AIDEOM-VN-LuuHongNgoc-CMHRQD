# Đồ án: Hệ thống AIDEOM-VN
Sinh viên: Lưu Hồng Ngọc - Trường Đại học Kinh tế (UEB - VNU)

## Hướng dẫn chạy hệ thống
1. Cài đặt thư viện: `pip install -r requirements.txt`
2. Chạy Dashboard: `streamlit run app.py`

## Cấu trúc Module
- m1_macro_forecast.py: Dự báo vĩ mô
- m2_digital_readiness.py: Xếp hạng TOPSIS
- m3_allocation_opt.py: Tối ưu phân bổ
- m4_labor_sim.py: Mô phỏng lao động
- m5_risk_eval.py: Đánh giá rủi ro đa mục tiêu
